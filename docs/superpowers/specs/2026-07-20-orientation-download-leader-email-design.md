# 오리엔테이션 자료 다운로드·조장 이메일 전달 설계

## 목표

기존 GNU AI Pioneer 캠프 안내 페이지에 오리엔테이션 PPTX 다운로드와 조장 이메일 전달 기능을 추가한다. 학생은 별도 계정 없이 모바일에서 자료를 내려받고, 조장 정보와 GPT 유료 계정 초대용 이메일을 운영총괄에게 보낼 수 있어야 한다.

## 확정된 방향

- 기존 홈페이지의 청색·주황색, 티켓·그리드 시각 체계를 유지한다.
- 이미 완성된 16:9 PPTX가 217KB로 충분히 작으므로 PDF를 중복 생성하지 않고 원본 PPTX만 제공한다.
- 페이지에 `자료·계정` 바로가기를 추가하고, 하나의 섹션 안에 `오리엔테이션 자료` 카드와 `조장 이메일 보내기` 카드를 배치한다.
- 이메일은 홈페이지 서버나 공개 결과물 목록에 저장하지 않는다. 조장이 조·이름·초대받을 이메일을 입력하고 개인정보 이용에 동의하면 운영총괄 `whoareyoukh@naver.com` 앞으로 제목과 본문이 채워진 메일 앱을 연다. 사용자가 메일 앱에서 보내기 버튼을 눌러야 전송이 완료된다는 안내를 명시한다.
- 새 참가자 제출코드는 `GNU-PIONEER-0721`로 교체한다. 코드는 공개 페이지에 노출하지 않고 운영진이 조장에게 전달한다.

## 접근 방식 비교

1. 메일 앱 직접 전달: 별도 외부 서비스나 개인정보 저장소가 필요 없고 운영총괄 받은편지함으로 바로 전달된다. 현재 배포 환경에서 가장 안전하고 즉시 사용할 수 있어 채택한다.
2. KV 저장 후 관리자 목록 제공: 브라우저 메일 앱이 없어도 수집되지만 관리자 화면과 다운로드 기능이 추가로 필요하고, 즉시 이메일 알림 요구를 충족하지 못한다.
3. Cloudflare Email Service 자동 발송: 완전 자동화할 수 있지만 현재 계정에 이메일 라우팅 도메인과 검증된 수신 주소가 없어 이번 일정에 바로 적용할 수 없다.

## 화면 구성

- 섹션 제목: `자료는 여기서 받고, 조장 이메일도 보내주세요.`
- 다운로드 카드: 파일 형식, 16:9, 22장, 파일 크기를 표시하고 `오리엔테이션 PPTX 다운로드` 버튼을 제공한다.
- 이메일 카드: 조 선택, 조장 이름, 초대받을 이메일, 개인정보 이용 동의 체크박스, `관리자에게 이메일 보내기` 버튼을 제공한다.
- 성공 상태: 메일 앱이 열렸음을 알리고 `받는 사람·제목·본문을 확인한 뒤 보내기`를 안내한다.
- 오류 상태: 누락값, 잘못된 이메일, 동의 미체크를 해당 입력요소 포커스와 함께 안내한다.

## 데이터 흐름과 개인정보

1. 입력값은 브라우저 메모리에서만 사용한다.
2. 스크립트가 `mailto:` 주소를 만들 때 모든 값을 URL 인코딩한다.
3. 이메일 본문에는 조, 조장 이름, 초대받을 이메일, 사용 목적만 포함한다.
4. 사이트, KV, 결과물 게시판, 로컬 저장소에는 이메일을 저장하지 않는다.

## 테스트와 완료 기준

- 정적 HTML 테스트가 다운로드 파일 경로, `download` 속성, 이메일 폼 필드, 개인정보 동의, 운영총괄 수신 주소를 검증한다.
- 스크립트 테스트가 이메일 검증, URL 인코딩, `mailto:` 생성, `innerHTML` 미사용을 검증한다.
- 빌드 테스트가 PPTX를 `dist`에 복사하고 원본과 크기가 같은지 검증한다.
- 전체 테스트와 빌드가 통과하고 공개 주소에서 파일 다운로드 응답과 페이지의 새 섹션이 확인되어야 한다.
