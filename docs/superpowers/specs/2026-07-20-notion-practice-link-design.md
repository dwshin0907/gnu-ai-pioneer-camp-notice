# 노션 실습페이지 링크 디자인

## 목표

학생이 홈페이지의 `자료·계정` 영역에서 노션 실습페이지를 즉시 열 수 있게 한다.

## 배치 선택

검토한 방식은 세 가지다.

1. 오리엔테이션 발표자료 카드 안에 두 번째 바로가기 버튼 추가
2. 자료 영역에 세 번째 독립 카드 추가
3. 자료 영역 제목 아래에 텍스트 링크 추가

첫 번째 방식을 사용한다. 실습에 필요한 자료가 한 카드에 모이고, 기존 2열 카드 구조와 모바일 레이아웃을 바꾸지 않아도 되며, 텍스트 링크보다 찾기 쉽다.

## 화면과 동작

- 기존 PPTX 다운로드 버튼 아래에 `노션 실습페이지 열기` 버튼을 추가한다.
- 보조 문구는 `실습 바로가기`로 표시한다.
- 주소는 `https://m.site.naver.com/2akZH`를 정확히 사용한다.
- 외부 페이지는 새 탭에서 열고 `rel="noopener noreferrer"`를 적용한다.
- 기존 GNU 파란색 카드 안에서 PPTX 다운로드 버튼과 구분되는 옅은 파란색 버튼으로 표시한다.
- 자료 영역 소개 문구에 노션 실습페이지가 포함되어 있음을 안내한다.

## 접근성과 반응형

- 링크 텍스트만으로 목적지를 이해할 수 있게 한다.
- 키보드로 접근 가능한 기본 앵커 요소를 사용한다.
- 기존 카드 폭과 반응형 그리드를 유지하고 버튼만 세로로 쌓는다.

## 검증

- HTML 테스트에서 링크 문구, 정확한 URL, 새 탭 열기와 안전 속성을 확인한다.
- 전체 테스트와 배포 빌드를 실행한다.
- 운영 배포 후 공개 페이지에서 링크가 실제로 포함됐는지 확인한다.
