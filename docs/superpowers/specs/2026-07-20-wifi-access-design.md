# 상단 Wi‑Fi 안내 설계

## 목표

캠프 안내 홈페이지 상단에서 학생이 연수원 Wi‑Fi 접속 정보를 즉시 확인할 수 있게 한다.

## 확정 구성

- 상단 고정 바로가기 메뉴의 첫 항목으로 `와이파이` 탭을 추가한다.
- 탭은 `#wifi`로 이동하며, 바로가기 메뉴 아래 첫 번째 콘텐츠로 Wi‑Fi 카드를 배치한다.
- 카드는 네트워크 ID `kb_sc`와 비밀번호 `kbsc4000`을 동일한 위계로 크게 표시한다.
- 카드에는 `강의실에서 연결할 때 사용하세요`라는 짧은 용도 안내를 넣는다.
- 기존 GNU 청색, 신호 주황색, 둥근 티켓 형태와 모바일 우선 레이아웃을 유지한다.
- 별도 JavaScript나 복사 기능은 추가하지 않는다. 접속 정보 확인이라는 요청 범위에 집중한다.

## 접근 방식 비교

1. 상단 탭 + 첫 화면 카드: 가장 빨리 찾을 수 있고 모바일에서도 동작이 단순하여 채택한다.
2. 준비물 섹션 안에 삽입: 구현은 쉽지만 현장에서 다시 찾기 어렵다.
3. 팝업 탭: 화면은 절약되지만 열기·닫기 동작과 JavaScript가 추가되어 현재 요청에는 과하다.

## 공개 범위

페이지 주소를 아는 사람은 Wi‑Fi 정보를 볼 수 있다. 사용자가 학생용 안내 홈페이지에 직접 게시하도록 요청했으므로 별도 로그인이나 마스킹은 적용하지 않는다.

## 완료 기준

- 상단 메뉴에 `와이파이` 링크가 존재한다.
- `#wifi` 영역에 정확한 ID와 비밀번호가 표시된다.
- 테스트와 빌드가 통과한다.
- 운영 주소에서 새 탭과 두 접속 정보가 확인된다.
