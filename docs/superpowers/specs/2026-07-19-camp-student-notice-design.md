# GNU AI Pioneer 캠프 학생 공지 페이지 설계

## 목적

2026년 7월 20일 출발하는 경상국립대학교 GNU AI Pioneer 캠프 1차 참가 학생이 문자로 받은 링크를 열고, 집결 시각과 장소, 준비물, 전체 일정, 긴급 연락처를 휴대폰에서 즉시 확인하게 한다.

## 핵심 사용자 경험

- 링크 미리보기와 첫 화면의 최우선 정보는 `7월 20일(월) 09:40까지 월계관(체육관) 앞 집결`이다.
- 버스는 10:00 정각 출발임을 집결 안내와 함께 반복한다.
- 집결 문의는 선탑자 김기현님 `010-6293-0916`으로 바로 전화할 수 있다.
- 학생은 스크롤하면서 준비물, 1일차·2일차 전체 타임테이블, 연수원 이용 규정, 복귀 안내를 순서대로 확인한다.
- 캠프 전 일정은 사천 KB손해보험 인재니움 사천연수원 안에서 진행하며 외부 이동이 없다는 점을 분명히 알린다.

## 정보 구조

1. 링크 공유용 메타데이터: 행사명, 집결 시각, 월계관 앞 집결 설명
2. 히어로: D-1 성격의 공지 라벨, 집결 시각, 장소, 10:00 정각 출발, 전화 버튼
3. 빠른 확인: 일정, 장소, 숙박, 전 일정 연수원 내 진행
4. 필수 준비물: 노트북과 충전기, 신분증, 세면도구, 개인 상비약
5. 전체 타임테이블: 1일차와 2일차의 모든 이동·교육·식사·체크아웃·수료 일정
6. 연수원 이용 규정: 명찰, 음식·주류 반입 금지, 식당 QR, 09:00 객실 퇴실
7. 복귀 및 연락: 7월 21일 16:00 연수원 출발, 학교 복귀 후 해산, 김기현님 연락처

## 시각 설계

- 색상: `GNU Blue #005BAC`, `Camp Blue #2563EB`, `Signal Orange #F97316`, `Ink #172033`, `Paper #F7FAFC`, `White #FFFFFF`.
- 서체: 시스템 한글 글꼴을 우선 사용해 외부 폰트 장애 없이 빠르게 표시한다. 큰 숫자와 시간을 굵은 산세리프로 강조하고 본문은 편안한 행간으로 읽게 한다.
- 레이아웃: 최대 760px의 단일 열 모바일 우선 구성이다. 데스크톱에서도 안내문처럼 집중된 폭을 유지한다.
- 시그니처: 첫 화면에 실제 버스 탑승권을 연상시키는 집결 패스를 배치한다. 절취선은 출발 전 확인해야 할 정보의 경계를 나타내는 기능적 장치다.
- 모션: 첫 화면의 제한적인 등장 효과만 사용하고 `prefers-reduced-motion`에서는 제거한다.

## 기술 설계

- 정적 `index.html`, `styles.css`, `script.js`로 구성한다.
- 일정과 공지 내용은 HTML에 직접 포함해 네트워크나 자바스크립트 실패에도 읽을 수 있게 한다.
- 자바스크립트는 준비물 체크 상태를 브라우저에 저장하는 점진적 기능만 담당한다.
- `tel:` 링크로 김기현님에게 바로 전화하며, 지도 링크는 외부 지도 서비스 종속을 피하기 위해 넣지 않는다.
- Cloudflare Pages 직접 배포와 GitHub 저장소 푸시를 모두 수행한다.

## 접근성 및 반응형

- 의미 있는 `header`, `main`, `section`, `ol`, `footer` 구조와 건너뛰기 링크를 사용한다.
- 텍스트 대비, 44px 이상의 터치 영역, 명확한 포커스 표시를 제공한다.
- 360px 모바일 화면에서 가로 스크롤 없이 표시하며, 760px 이상에서는 일정 카드가 시간 열과 내용 열로 정렬된다.
- 자바스크립트가 꺼져 있어도 모든 필수 정보가 보인다.

## 콘텐츠 정확성 기준

- 날짜는 2026년 7월 20일 월요일부터 7월 21일 화요일까지다.
- 집결 마감은 09:40, 버스 출발은 10:00이다.
- 집결지는 경상국립대학교 가좌캠퍼스 월계관(체육관) 앞이다.
- 캠프 장소는 사천 KB손해보험 인재니움 사천연수원, 강의장은 402호다.
- 참가자 숙박은 전원 1인 1실이며 식사는 4식이다.
- 선탑자 김기현님의 전화번호는 010-6293-0916이다.
- 내부 운영진 이메일, 기사 연락처, 연수원 담당자 연락처 등 학생에게 불필요한 개인정보는 공개하지 않는다.

## 검증

- 자동 검증으로 필수 문구, 모든 시간대, 전화 링크, 메타데이터, 의미 구조를 확인한다.
- HTML 정적 검사와 로컬 서버 응답 검사를 실행한다.
- Chromium에서 390px 모바일과 1440px 데스크톱 화면을 캡처해 레이아웃과 가로 넘침을 확인한다.
- Cloudflare 배포 후 공개 URL의 상태 코드와 핵심 문구를 다시 확인한다.
